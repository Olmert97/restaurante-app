# Sistema de Gestión de Restaurante - UNAH 2026

Este es el proyecto para la clase de **Programación Avanzada**. 
Contexto del Problema
Una cadena de restaurantes con múltiples sucursales necesita un sistema integral para gestionar operaciones diarias: desde el control de menú y inventario hasta la toma de pedidos, gestión de mesas, facturación y análisis de ventas. El sistema debe permitir operación en tiempo real, gestión de múltiples meseros, y generar reportes para toma de decisiones administrativas.

## Estructura del Proyecto
- `/config`: Archivos de configuración y conexión a la DB.
- `/database`: Scripts SQL y diagramas.
- `/docs`: Documentación del proyecto.
- `/views`: Interfaces de usuario (HTML/PHP).

## Tecnologías utilizadas
* **Lenguaje:** PHP 8.x
* **Base de Datos:** MySQL
* **Servidor Local:** XAMPP
* **Control de Versiones:** Git & GitHub
